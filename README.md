Software Yang Digunakan :
-------------------------
1. Browser (Google Chrome)
2. XAMPP (Lokal Web Server)
3. Text Editor (Atom)

Cara Menjalankan Aplikasi
-------------------------
1. Secara Offline dapat menggunakan Web Server (Xampp) untuk menjalankan program dengan cara mengaktifkan module Apache dan MySQL,  kemudian membuat dan memasukan file direktori pada folder HTDOCS setelah itu menjalankannya dengan server lokal Browser (Google Chrome).
2. Secara online dapat melalui link berikut https://www.onlinegdb.com/ dan  https://es6console.com/ dengan mengcopy sourcecode dari text editor

Bahasa Pemrograman Yang digunakan PHP dan Javascript
-------------------------------------------------------------------------------------------------------------------------------------------
Soal No 1 : 
Fungsi JSON (JavaScript Object Notation) untuk pertukaran dan penyimpanan data aplikasi dari web API (server) ke client atau sebaliknya
